Archive.ninja
Archive.ninja uses javascript to open multiple windows to multiple archive sites, so you need to allow popups in your browser for archive.ninja.
We don't rely on any third party javascript or resources and our code is available on github.
To use archive.ninja, we recommend using a bookmarklet, which is a small bit of javascript that runs from your bookmarks bar. Copy and paste this code into a new bookmark on your bookmarks bar.
